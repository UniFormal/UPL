Nodes are Theories.
Yellow nodes are not formalized in set theory. Purple nodes are formalized in set theory.

Arrows:
____> (normal arrows): include, points towards the theory that includes.
----> (dashed arrows): realize, points towards the theory that realizes.
|____> (thick arrow): views, points towards the theory that is realized with the view.
|----> (thick dashed arrows): structure or total structure, points towards the theory formalizing the structure.
|_-_-> (thick black arrow): structure view is used.
